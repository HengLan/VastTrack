# VastTrack: Vast Category Visual Object Tracking

<img src="https://github.com/HengLan/VastTrack/blob/main/assets/VastTrack-img.png" width="110%">

**Figure:** We introduce **VastTrack**, a new large-scale benchmark that aims at facilitating general single object tracking with abundant object categories (2.1K classes) and videos (over 50K sequences). Here we show the partial target trajectory in a video. <em>Please note that, only a very small part of categories and videos are displayed</em>. More details can be seen in the [**paper**](https://arxiv.org/abs/2403.03493).
